## Beecrowd Solutions 
This repository contains my accepted solutions to problems from Beecrowd (formely URI Online Judge).

The goal of this repository is to practice algorithms, problem-solving, and competitive programming using C++ and Java.

## Repository Structure
```
Beecrowd-Solutions/
│
├── cpp/
│   ├── 1000.cpp
│   ├── 1001.cpp
│   └── ...
│
├── java/
│   ├── 1000.java
│   ├── 1001.java
│   └── ...
```
cpp/ → Solutions implemented in C++

java/ → Solutions implemented in Java

Each file is named according to the problem ID from Beecrowd.

 Improve logical reasoning
